yadism.input.inspector module¶

The purpose of this module is to provide a runner that implements the semantics for the input restrictions defined in the following files:

  • domains.yaml: in which all the domains restrictions are defined

  • cross_constraints.yaml: in which further restrictions are defined,

    each one involving more than one input field

class yadism.input.inspector.EnumArgument(**kwargs)[source]¶

Bases: yadism.input.inspector.Argument

Argument definition for enum type.

enum type is just a domain defined by enumerating all the possible values. There is no hard-coded restrictions in having in the same enum domain values from different default types (e.g. integers and strings may be mixed).
